# 데이터를JSON 파일로 관리하는CRUD(Create, Read, Update, Delete) 콘솔 애플리케이션을 개발

기존에 진행했던 PoC(Proof of Concept) 코드의 구조와 로직을 참고하여, 
데이터를 JSON 파일로 관리하는 CRUD(Create, Read, Update, Delete) 콘솔 애플리케이션을 개
발한다.


## 요구사항
- PoC에서 사용된코드구조를유지한상태로, CRUD 구현

### CRUD 기능 구현
- Create: 새로운 데이터를 입력 받아 JSON 파일에 저장. 
- Read: 전체 목록 보기 및 특정ID/키 값으로 검색 기능. 
- Update: 기존 데이터를 선택하여 특정 필드 수정. 
- Delete: 특정 데이터를 안전하게 삭제.
